# Porto - Rifqi Fahrezi Portfolio

Welcome to the source code of my personal portfolio: a space where game design instincts, Flutter craft, and part-time Teelite product work blend into cinematic web experiences.

---

## Highlights

- React + Vite + TypeScript foundation styled with Tailwind and animated through Framer Motion.
- Selected work spans Teelite's e-commerce platform, finance tracker, Exorcist Bane prototyping, and student innovation programs.
- Motion system respects user accessibility preferences while staying smooth and expressive.
- Content-driven architecture keeps copy and data in a single source of truth.

---

## Project Structure

| Path | Description |
| ---- | ----------- |
| `src/components/` | Section components (Hero, Work, About, Process, Contact). |
| `src/data/content.ts` | Centralized navigation, work, principles, experience, and contact copy. |
| `src/lib/motion.tsx` | Framer Motion helpers and provider wrapper. |
| `public/` | Static assets, favicons, imagery. |

---

## Getting Started

```bash
# Clone
git clone https://github.com/your-username/porto.git
cd porto

# Install dependencies
npm install

# Run the dev server
npm run dev

# Build for production
npm run build
```

The dev server runs on http://localhost:5173.
